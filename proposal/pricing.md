# Pricing

We propose a fixed-fee structure for the required scope, with desired items priced
separately and delivered only as capacity allows.

## Required scope (R1–R4) — fixed fee

| Phase | Scope | Fee |
|---|---|---|
| 1. Architecture review | R4 | $10,000 |
| 2. Reports remediation | R1, partial R3 | $18,000 |
| 3. Restocking build | R2, partial R3 | $42,000 |
| 4. Test coverage close-out | Remaining R3 | $10,000 |
| **Subtotal** | | **$80,000** |

This is a not-to-exceed price for R1–R4. If scope changes materially once the
architecture review is complete, for example if the audit surfaces issues well beyond
Reports' known defects, we'll flag it before proceeding and re-scope by change order
rather than absorbing it silently or billing past the ceiling without agreement.

## Desired scope (D1–D3) — time and materials, not-to-exceed

| Item | Not-to-exceed |
|---|---|
| D1. UI modernization | $12,000 |
| D2. Internationalization (remaining modules) | $6,000 |
| D3. Dark mode | $8,000 |
| **Subtotal** | **$26,000** |

Billed only for time actually spent, and only pursued in priority order as time
allows within the Phase 5 window described in the timeline. Meridian is not obligated
to fund all three.

## Total

- **Required scope:** $80,000 fixed fee
- **Desired scope (if fully pursued):** up to $26,000 T&M, not-to-exceed
- **Combined ceiling:** $106,000

This falls within the $70K–$120K range discussed, with room held back deliberately so
that pursuing all three desired items doesn't require a change order.
